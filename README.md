# CFP backend (Flask, Mongoengine, JWT)

## Backend Setup

### Create folder and activate it for packages
```
cd backend
python -m venv venv
venv\Scripts\activate
```

### Package install
```
pip install -r requirements.txt 
```

## Backend Run
```
Flask run
```

## Frontend Setup and Run

```
yarn
yarn start
```
or 
```
npm install
npm start
```
